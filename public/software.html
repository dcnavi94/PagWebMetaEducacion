Ingeniería en Software

Desarrolla las apps y sistemas que moverán el mundo del mañana.

3 años Cuatrimestral
Agenda una Cita
RVOE SEG

Programa con Reconocimiento de Validez Oficial de Estudios avalado por la Secretaría de Educación de Guanajuato.

Objetivo de la Carrera

Formar ingenieros visionarios que dominen los lenguajes, frameworks y entornos cloud más solicitados globalmente. Queremos pioneros que creen sistemas escalables, seguros y rentables usando Inteligencia Artificial y Cloud Computing.

Perfil de Egreso

El egresado poseerá altos dotes para el análisis lógico y matemático, además manejará sólidas competencias técnicas de front-end y back-end (fullstack). Liderará células de desarrolladores manejando metodologías ágiles en ambientes corporativos reales.

Campo Laboral

  • Desarrollador de Aplicaciones Móviles / Web Full-Stack
  • Ingeniero de Datos y Modelos de Aprendizaje Automático
  • Arquitecto Cloud de Soluciones Corporativas AWS/Azure
  • Scrum Master, Product Owner o Technical Lead en TI

Modalidades

  • Presencial Intensiva: Lunes a Viernes. Prácticas en todos los laboratorios para enfoque total.
  • Presencial Sabatina / Virtual: Ideal si trabajas, puedes estudiar desde el sábado de forma presencial intensiva o todo el contenido vía virtual.

Certificaciones Incluidas

AWS Cloud Practitioner Microsoft Azure Scrum Developer
